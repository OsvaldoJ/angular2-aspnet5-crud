# angular2-aspnet5-crud

This progressive web app is composed of 2 pages, the home page and a simple CRUD page.

**BUILT WITH**

 - Angular 2
 - ASP.NET Core
 - Angular Universal
 - Progressive Web Apps
 - Webpack
 - Grunt
